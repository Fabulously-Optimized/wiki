# Mod Menu

1. With the game open, click `Mods`
2. Scroll down to the specific mod you want to disable, note the third row with the ⚒️ (tools) or ✏️ (pencil) icon.

{% tabs %}
{% tab title="Icon is blue" %}

If the text mentions **"Toggle mod"**, click the config button ![config](https://i.ibb.co/j35cBtn/image.png) and find the setting that disables the mod. Otherwise config button ![config](https://i.ibb.co/j35cBtn/image.png) and see if you can disable the specific feature you need.

{% endtab %}

{% tab title="Icon is gray" %}

Either:

* there is no config option to disable the mod
* it is not easy to disable the mod from in-game

In that case, [follow the launcher instructions](../) for disabling the mod instead.

{% endtab %}

{% tab title="No icon on the row" %}

You added the mod yourself. Check if there are config options ![config](https://i.ibb.co/j35cBtn/image.png) and if so, whether the mod has a way to disable itself.

Otherwise, [follow the launcher instructions](../) for disabling the mod instead.

{% endtab %}

{% tab title="No icons at all" %}

You don't have the Mod Menu Helper resource pack enabled for some reason.

1. Click `Done`
2. Click `Options...`
3. Click `Resource Packs...`
4. Click `⏵` on "Mod Menu Helper.zip"
5. Click `Done`
5. Click `Mods`
6. Now look again to see which option matches best.

{% endtab %}
{% endtabs %}
